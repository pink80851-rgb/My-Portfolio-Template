🔧 問題 (Problem)

手動管理 Fortigate VIP 規則很重複，容易出錯，特別是在處理多個服務和 IP 映射時。

🚀 解決方案 (Solution)

本專案自動化 Fortigate VIP 設定，減少手動操作的負擔，降低錯誤率。

⚙️ 功能 (Features)
自動建立 VIP
減少重複的防火牆設定操作
簡化 IP 映射流程
提升操作效率
🧠 SRE 概念 (SRE Concept)
降低重複勞務 (Reduce toil)
自動化操作 (Automation)
基礎設施即程式化 (Infrastructure as Code 的概念)
🛠 技術 (Tech)
Bash / Python（依你實際使用填寫）
Fortigate CLI / API
